## Operational Standards

This repository contains the core architecture for the Hanlan Group institutional platform.

## Local Development

**Prerequisites:** Node.js (Latest LTS recommended)

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Environment Configuration:**
   Create a `.env.local` file and specify the required parameters:
   ```env
   GEMINI_API_KEY="YOUR_API_KEY"
   ```

3. **Launch Development Server:**
   ```bash
   npm run dev
   ```
